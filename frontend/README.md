# Barbershop Frontend

Next.js 15 frontend for the Barbershop Booking System.

See the full project documentation in the root [`README.md`](../README.md).

## Quick Start

```bash
npm install
npm run dev -- --port 9000
```

Requires the backend to be running on `http://localhost:5000`.
Set `NEXT_PUBLIC_API_URL` in `.env.local` if the backend is on a different port.
